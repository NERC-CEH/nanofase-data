Welcome to NanoFASE Data's documentation!
=========================================

.. toctree::
   :maxdepth: 2
   :caption: Contents:

The NanoFASE Data scripts are responsible for generating input data for the `NanoFASE model <|nfurl|>`_, by compiling input datasets or constants for each required input variable into one NetCDF file.

.. Insert schematic image of nanofase data

This documentation will guide you in how to use the NanoFASE Data scripts with your own separate input datasets, to generate your own input data file.
